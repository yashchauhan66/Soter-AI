# Phase 10 Outreach Campaign

## Goal and Offer

Core offer: a free, permissioned AI chatbot security audit using safe synthetic tests.

CyberRakshak Guard helps chatbot and RAG builders detect, block, redact, monitor, and report prompt injection, sensitive-data exposure, unsafe responses, and private-document risk. It includes Guard APIs, an SDK, reports, a security badge, and agency white-label workflows. It is an OWASP LLM Top 10 aligned defense-in-depth layer, not a security guarantee.

## Send Controls

1. Verify the company, recipient, role, and public contact channel.
2. Confirm that the opening sentence is based on public, current information.
3. Never imply that a vulnerability was found before an authorized audit.
4. Send no more than two follow-ups unless the recipient provides a future date.
5. Record consent, opt-out, reply, demo, and next action in the lead tracker.
6. Never run tests against a target without written authorization.
