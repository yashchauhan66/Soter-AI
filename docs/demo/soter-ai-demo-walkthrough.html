🛡️ Input Guard — Before LLM

Inspects every user message before it reaches your LLM. Detects prompt injection, jailbreaks, system prompt leak attempts, PII, secrets, and token abuse.

🔍 Detection Capabilities

  • Prompt injection detection (200+ regex patterns)
  • Jailbreak / DAN / role-play bypass detection
  • System prompt extraction attempts
  • Encoding obfuscation (base64, hex, morse)
  • Multilingual attacks (Hindi, Chinese, Russian, Arabic)
  • Leetspeak / homoglyph bypass patterns
  • Token abuse / large payload detection

📝 Example: Using REST API

curl -X POST http://localhost:3000/api/guard/input \
  -H "Content-Type: application/json" \
  -H "x-api-key: ck_test_your_key" \
  -d '{"message":"Ignore previous instructions"}'

Returns: BLOCKED with findings and risk score

4

🛡️ Output Guard — After LLM

Inspects LLM responses before sending to the user. Prevents data leakage, unsafe content, system prompt leakage, and suspicious URLs.

🔍 Detection Capabilities

  • System prompt leakage in output
  • Unsafe / harmful content detection
  • Suspicious URL / spam link detection
  • PII and secrets in model responses
  • Fallback message on block

📝 Example: Using TypeScript SDK

import { Soter } from "@soterai/core";

const soter = new Soter({
  apiKey: process.env.SOTER_API_KEY
});

const result = await soter.protect({
  input: userMessage
});

if (!result.allowed) {
  console.log("Blocked:", result.reason);
}

🎯 Intent Guard

Verifies that every agent action matches the original user intent. Blocks out-of-scope operations.

How It Works

  • Extract user intent from the original prompt
  • Define allowed and forbidden intent categories
  • Check each action against the extracted intent
  • Score intent-match on every tool call
  • Decide: ALLOW, BLOCK, ASK_APPROVAL, or REVIEW

Intent Categories

code_execution, data_reading, data_writing, data_deletion, payment, communication, authentication, admin, system_config, external_access, file_operations, web_browsing, search, analysis, content_creation

📦 JavaScript / TypeScript SDK

@soterai/core — The primary NPM package for Node.js and browser-based integrations.

Installation

npm install @soterai/core

Quick Start

import { Soter } from "@soterai/core";

const guard = new Soter({
  apiKey: process.env.SOTER_API_KEY,
  baseUrl: process.env.SOTER_BASE_URL
});

// Protect a chat interaction
const result = await guard.protectChat({
  message: userMessage,
  callLLM: async (safeMsg) => myLLM(safeMsg)
});

if (result.allowed) {
  console.log("Response:", result.safeResponse);
}

Available Methods

  • guard.input() / guard.guardInput() — Check user input
  • guard.output() / guard.guardOutput() — Check model output
  • guard.protectChat() — Full chat protection (input → LLM → output)
  • guard.protectRag() — RAG pipeline protection
  • guard.secureChat() — Simplified secure chat
  • guard.startAgentSession() — Start agent firewall session
  • guard.checkAgentAction() — Check agent tool calls
  • guard.checkDataLeak() — Check data for exfiltration
  • guard.wrapTool() — Wrap tools with automatic guard
  • guard.scanMcpTools() — Scan MCP server tools
  • guard.checkBrowserForm() — Browser form safety check
  • guard.checkMemory() — Agent memory check
  • guard.createCanary() — Deploy canary tokens
  • guard.checkCanaryLeak() — Check for canary leaks
  • + 20+ more methods for advanced security features
