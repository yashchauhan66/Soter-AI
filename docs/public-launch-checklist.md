# Public Launch Checklist

- [ ] Production readiness blockers reviewed and assigned.
- [ ] Terms and privacy reviewed by counsel.
- [ ] No fake logos, testimonials, certifications, or benchmark claims.
- [ ] Home, pricing, demo, agency, enterprise, pilot, case study, contact, changelog, status, trust, and docs links checked.
- [ ] Limitations and defense-in-depth wording visible.
- [ ] Beta onboarding and support ownership staffed.
- [ ] Live billing validation complete.
- [ ] Status page and incident drill complete.
- [ ] Backups restored in staging.
- [ ] Monitoring alerts reach the on-call owner.
- [ ] Demo content contains no real customer data.
- [ ] Launch analytics contain metadata only, never raw sensitive content.
