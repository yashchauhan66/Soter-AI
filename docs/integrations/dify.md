# Soter Guard — Dify Plugin Guide

## Overview

Dify plugin with YAML tool definitions for Input Guard, Output Guard, PII Redactor, and RAG Scanner.

## Quick Start

1. Upload the `packages/integrations/dify/` directory as a Dify plugin
2. Configure your Soter API key in the provider credentials
3. Use Soter tools in your Dify workflows

See [Dify README](../../packages/integrations/dify/README.md) for details.
