## 1. BROKEN ITEMS

### TypeScript Typecheck (`npx tsc --noEmit`)
- **Status**: ❌ FAILS
- **Error Count**: ~120+ errors
- **Root Cause**: ALL errors are `TS6053: File not found` for `.next/types/**/*.ts`
- **Reason**: The `.next/types` directory is generated by `next build` but does not exist after a clean clone or after `.next` was gitignored/deleted
- **Impact**: LOW — This is NOT a code error. Running `next build` first generates these types and tsc passes
- **Fix**: Run `npm run build` before `npm run typecheck`, or exclude `.next/types` from tsconfig

### Next.js Lint (`npx next lint`)
- **Status**: ⚠️ DEPRECATED
- **Root Cause**: `next lint` is deprecated in Next.js 15 in favor of running ESLint directly
- **Impact**: LOW — Linting infrastructure exists but the command is outdated
- **Fix**: Use `npx eslint .` directly or update the npm script

### Docker Compose Config (`docker compose -f docker-compose.prod.yml config`)
- **Status**: ❌ FAILS
- **Error**: `required variable POSTGRES_PASSWORD is missing a value`
- **Root Cause**: `docker-compose.prod.yml` interpolates `${POSTGRES_PASSWORD}` from environment, but no `.env` file has it set
- **Impact**: MEDIUM — Docker deployment won't work without env vars
- **Fix**: Set `POSTGRES_PASSWORD` in environment or `.env.production`

### Docker Build
- **Status**: ❌ NOT TESTED (blocked by config failure above)
- **Impact**: MEDIUM

### E2E Playwright Tests
- **Status**: ❌ TIMEOUT
- **Error**: `Timed out waiting 120000ms from config.webServer`
- **Root Cause**: Port 3000 was already in use by another process
- **Impact**: MEDIUM — E2E tests never ran
- **Fix**: Kill existing processes on port 3000, or change Playwright config port

### Python SDK Async Tests
- **Status**: ❌ 6 TESTS FAIL
- **Failing Tests**:
  - `test_async_input_sends_api_key`
  - `test_async_analyze_no_auth`
  - `test_async_protect_chat_blocks_injection`
  - `test_async_protect_chat_allows_safe_with_async_llm`
  - `test_async_protect_chat_accepts_sync_llm`
  - `test_async_protect_rag_excludes_risky`
- **Root Cause**: Missing `pytest-asyncio` plugin dependency
- **Impact**: MEDIUM — Async client functionality untested
- **Fix**: Add `pytest-asyncio` to `packages/python-sdk/pyproject.toml` dev dependencies
