# Short Demo Video Script

Target length: 75-90 seconds.

## Scene 1: Problem (0-12s)

Customer-facing AI can receive prompt injection, personal data, secrets, and unsafe instructions. Model safety alone does not show your team what happened.

## Scene 2: Product Flow (12-35s)

Show the application sending input to CyberRakshak Guard, then the model, then output guard. On screen: detect, block, redact, monitor, report.

## Scene 3: Evidence (35-60s)

Show a synthetic injection blocked, fictional PII redacted, signed alert, and report. For RAG, show a risky document quarantined.

## Scene 4: Audience (60-75s)

Show SaaS, agency, and enterprise pilot paths. No customer logos unless approved.

## Scene 5: Close (75-90s)

CyberRakshak Guard is an OWASP LLM Top 10 aligned AI security gateway for chatbots, RAG apps, and agents. Try the demo or request an authorized security audit. Defense-in-depth reduces risk; no control is absolute.
