# Product Demo Script

1. State the honest scope: risk reduction, not complete protection.
2. Show a safe request through the input guard.
3. Show a synthetic prompt-injection example.
4. Show fictional PII/secret redaction.
5. Show output inspection and fallback.
6. Open logs, policy, signed alert, report, and badge.
7. For RAG, show quarantine, retrieval isolation, and no-source fallback.
8. For agencies, show client separation and white-label reporting.
9. Close with integration effort, plan, limitations, and next step.

Never use production credentials, customer logs, or unauthorized targets.
