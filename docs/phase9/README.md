## Operating Rules

- Use honest OWASP LLM Top 10 aligned, defense-in-depth, and risk-reduction positioning.
- Never claim complete protection, certification, fake traction, or invented ROI.
- Run defensive audits and red-team scenarios only with owner authorization.
- Store no raw secrets, unredacted customer logs, or sensitive content in CRM/outreach tools.
- Require written approval before publishing customer identity, logo, quote, evidence, or metrics.
- Add product scope only when repeated evidence from qualified buyers proves the need.
